Optical LAN Training: Operations, Provisioning & Maintenance

Will you be responsible for Provisioning and Maintaining a Tellabs Optical LAN?

This course is designed for Network Operations and/or any individuals responsible for operating, provisioning, maintaining and monitoring a Tellabs Optical LAN. This course is also intended for employees responsible for the administration, installation, testing, maintenance or technical support of the Optical LAN.

Prerequisites

Basic knowledge in Ethernet switching and routing; including VLAN concepts, IP/subnetting, basic QoS knowledge and the basic concepts of port security.

Class Specifics

Course Duration: 3 days

Class Location: Public Classroom, Virtual (On-Line), Private Classroom

Class Size: 4-8 Students

Day 1 – Optical LAN System Architecture and Applications

Overview

This instructor-led course provides an overview of the Tellabs Optical LAN system. It is an introduction to the network architecture and applications.

After a brief review of the features and functionality of the Tellabs Optical LAN solution, students will learn about the applicable technology, architecture, and hardware functionality.

Course Objectives

Upon completion of this course, students will be able to:

  1. Identify the features and functionality of the Tellabs Optical LAN Solution

  2. Describe the network functionality of the Tellabs Optical LAN Solution

  3. Describe the common equipment associated with the Tellabs Optical LAN Solution equipment series

Day 2-3 – Optical LAN Operations, Provisioning and Maintenance

Overview

The instructor-led course provides knowledge and skills needed to operate, provisioning, and maintain a Tellabs Optical LAN system using the Tellabs Panorama PON Element Management System (EMS).

After a brief introduction to the Tellabs Optical LAN system and hardware functionality, students will learn how to provision the system to provide the voice, data, and video services and application supported by the Tellabs Optical LAN system.

This course includes hands on activities using Tellabs training lab equipment. It is designed to help students learn the concepts that have been presented and reinforce what they have learned.

Course Objectives

Upon completion of this course, students will be able to:

  1. Perform provisioning procedures for the Tellabs Optical LAN voice and data services using Panorama PON.

  2. Perform equipment management and monitoring procedures using the Tellabs Panorama PON EMS

  3. Perform trouble clearing procedures of common problems

Course Outline

EMS Operations

  • Navigate the Panorama PON Network Manager
  • Add and configure the Network Element
  • Configure links, ONTs, and Ports
  • Panorama Overview
  • NE Management

Service Provisioning

  • Create Customer Services
  • Data Service Creation
  • CLI Commands for Provisioning

Maintenance

  • Manage System Alarms and Status
  • Perform OLT Field Maintenance
  • Perform Maintenance Procedures
  • Using the Alarms and Events View
  • Using the Troubleshooting, Test, and Maintenance User’s Guide
  • Line Status
  • Tellabs Optical LAN Craft Interface
  • EMS Maintenance
  • Creating Reports
